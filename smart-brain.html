001 : Smart-Brain API

A full stack image recognition app created while taking The Complete Web Developer Course: Zero to Mastery with instructor Andrei Neagoie.

User can choose to register/sign-in. When signed-in, images can be uploaded and a face detection function will detect faces/portraits in photograph, adding 1 point to user ranking.
