002 : Weather-App

React application that can detect weather for the next 7 days, including pressure percentage and chart variation.

A window will pop up on the screen asking for location detection to be on. If approved, function will be able to fetch openweather API and display current weather. There is also an option to manually type out city, state & zip.

Made With: React, API
